# Simple Gemini Chat (GitHub Pages)

Tek ekranlik basit bir chat arayuzu.

## Kurulum

1. Siteyi ac.
2. Ilk acilista uygulama Gemini API key ister.
3. Her kullanici kendi key degerini kendi tarayicisina girer.

## Not

Bu proje tamamen istemci tarafinda calisir. API key kullanicinin kendi tarayicisinda localStorage icinde tutulur.
Her kullanici yalnizca kendi key degerini kullanir.
